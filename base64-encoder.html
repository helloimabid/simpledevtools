What is Base64?

Base64 is an encoding scheme that converts binary data into an ASCII string format. It's commonly used to encode data that needs to be transmitted over media designed to handle text.

Common use cases include embedding images in HTML/CSS, encoding credentials, and transmitting data in JSON or XML.
