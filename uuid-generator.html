What is a UUID?

A UUID (Universally Unique Identifier) is a 128-bit value used to uniquely identify information. Version 4 UUIDs are randomly generated.

Format: xxxxxxxx-xxxx-4xxx-yxxx-xxxxxxxxxxxx (where x is any hexadecimal digit and y is one of 8, 9, A, or B)

Use cases: Database primary keys, session identifiers, file names, API request IDs.
