1. Regex Basics

Regular expressions (regex) are patterns used to match character combinations in strings. They're used for validation, search/replace, and text extraction.

Key Concepts:

  • Literal characters match themselves: cat matches "cat"
  • Special characters have special meaning: . * + ? ^ $ [ ] { } ( ) | \
  • Escape special chars with backslash: \. matches a literal period

2. Character Classes

Pattern Description Example
. Any character (except newline) a.c ‚Üí abc, a1c, a@c
\d Digit [0-9] \d{3} ‚Üí 123, 456
\D Non-digit \D+ ‚Üí abc, @#$
\w Word character [a-zA-Z0-9_] \w+ ‚Üí hello_123
\W Non-word character \W ‚Üí @, #, space
\s Whitespace (space, tab, newline) a\sb ‚Üí a b
\S Non-whitespace \S+ ‚Üí hello
[abc] Any of a, b, or c [aeiou] ‚Üí vowels
[^abc] Not a, b, or c [^0-9] ‚Üí non-digits
[a-z] Range a to z [A-Za-z] ‚Üí letters

3. Quantifiers

Pattern Description Example
* 0 or more a* ‚Üí "", a, aa, aaa
+ 1 or more a+ ‚Üí a, aa, aaa (not "")
? 0 or 1 (optional) colou?r ‚Üí color, colour
{n} Exactly n times \d{4} ‚Üí 2024
{n,} n or more times \d{2,} ‚Üí 12, 123, 1234
{n,m} Between n and m times \d{2,4} ‚Üí 12, 123, 1234
*? Non-greedy * (match minimum) <.*?> ‚Üí first tag only
+? Non-greedy + \d+? ‚Üí single digit

4. Anchors & Boundaries

Pattern Description Example
^ Start of string/line ^Hello ‚Üí starts with Hello
$ End of string/line world$ ‚Üí ends with world
\b Word boundary \bcat\b ‚Üí "cat" not "cats"
\B Non-word boundary \Bcat ‚Üí "scat" not "cat"

5. Groups & Lookaround

Pattern Description Example
(abc) Capture group (ha)+ ‚Üí haha
(?:abc) Non-capture group (?:ha)+ ‚Üí match but don't capture
(?=abc) Positive lookahead \d(?=px) ‚Üí digit before px
(?!abc) Negative lookahead \d(?!px) ‚Üí digit not before px
(?<=abc) Positive lookbehind (?<=\$)\d+ ‚Üí digits after $
(?<!abc) Negative lookbehind (?<!\$)\d+ ‚Üí digits not after $
a|b Alternation (or) cat|dog ‚Üí cat or dog
\1 Backreference (.)\1 ‚Üí repeated char (aa, bb)

7. Regex in JavaScript

// Create regex
const regex = /pattern/flags;
const regex2 = new RegExp('pattern', 'flags');

// Flags
// g = global (find all matches)
// i = case-insensitive
// m = multiline (^ and $ match line start/end)

// Test if matches
regex.test('string');  // returns true/false

// Find matches
'string'.match(/pattern/g);  // returns array or null
[...'string'.matchAll(/pattern/g)];  // all matches with groups

// Replace
'hello world'.replace(/world/, 'regex');  // 'hello regex'
'hello world'.replace(/o/g, '0');  // 'hell0 w0rld'

// Split
'a,b;c'.split(/[,;]/);  // ['a', 'b', 'c']

// Extract groups
const match = 'John Doe'.match(/(\w+) (\w+)/);
// match[0] = 'John Doe' (full match)
// match[1] = 'John' (first group)
// match[2] = 'Doe' (second group)
