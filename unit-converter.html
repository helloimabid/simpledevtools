Length Quick Reference

1 kilometer = 0.621 miles

1 mile = 1.609 kilometers

1 meter = 3.281 feet

1 inch = 2.54 centimeters

Data Storage Quick Reference

1 KB = 1,024 Bytes

1 MB = 1,024 KB

1 GB = 1,024 MB

1 TB = 1,024 GB

Temperature Quick Reference

0°C = 32°F = 273.15K

100°C = 212°F = 373.15K

°F = (°C × 9/5) + 32

°C = (°F - 32) × 5/9

Weight Quick Reference

1 kilogram = 2.205 pounds

1 pound = 0.454 kilograms

1 ounce = 28.35 grams

1 stone = 6.35 kilograms

How to Use

  1. Select a unit category (Length, Weight, Temperature, etc.)
  2. Enter the value you want to convert in the "From" field
  3. Select the source unit from the dropdown
  4. Select the target unit from the "To" dropdown
  5. The result appears instantly with high precision
  6. Use the swap button to reverse the conversion
